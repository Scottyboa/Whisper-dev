This new info button shows ongoing status messages and updates for the web app (such as new features, bug fixes, and important notifications).

AWS time limit update:
Until now, if a note generated via the AWS models took more than 45 seconds, an error could occur. This has now been fixed by increasing the time limit to 2.5 minutes, which should be more than enough to generate a note.

To apply this update, you need to do the following: Go to the front page, click the “AWS guide” button next to the AWS Bedrock key input fields, scroll all the way to the bottom of the page, and follow the instructions under “HOW TO UPDATE THE STACK (CLOUDFORMATION)”. Once this is done, the issue will be resolved.
