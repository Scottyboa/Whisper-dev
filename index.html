How to Sign Up for an OpenAI API Account and Use It for Transcription

1. Creating an OpenAI API Account

To use the transcription tool at Whisper Transcription, you need an OpenAI API key. Follow these steps to create an account and obtain your API key.

Step 1: Sign Up for OpenAI

  • Go to OpenAI's website.
  • Click Sign Up in the top-right corner.
  • Enter your email address, create a password, and verify your email.
  • Follow the on-screen instructions to complete your profile.

Step 2: Access the API Key

  • Once logged in, navigate to the API section at OpenAI Platform.
  • Click on "View API Keys" under your account settings.
  • Click "Create new secret key".
  • Copy the generated API key and save it in a secure location.

âš  Important: You will not be able to see this key again after closing the page!

2. Purchasing API Credits (Tokens)

To use OpenAI's Whisper transcription model, you need to buy API credits.

Step 1: Add a Payment Method

  • Go to OpenAI Billing.
  • Click "Manage Billing" and add a credit or debit card.
  • Set up a spending limit to control costs if needed.

Step 2: Buy Tokens

  • Open the Usage & Limits section.
  • Select "Add Funds" to purchase credits.
  • Choose an amount and complete the payment.

âœ… Your account is now ready to use for transcriptions!

3. Entering Your API Key in the Whisper Transcription Tool

Once you have an API key, you need to enter it into the transcription tool to start using it.

Step 1: Open the Transcription Page

  • Go to Whisper Transcription.
  • Find the field labeled "Enter API Key".

Step 2: Paste Your API Key

  • Copy your API key from your saved location.
  • Paste it into the API Key field on the Whisper page.
  • Click "Enter Transcription Tool".

âœ… Your transcription tool is now ready to use!
